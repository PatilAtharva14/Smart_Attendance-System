# 🎓 Smart AI Attendance System using Face Recognition

![Python](https://img.shields.io/badge/Python-3.11-blue)
![Flask](https://img.shields.io/badge/Flask-Web_Framework-black)
![OpenCV](https://img.shields.io/badge/OpenCV-Computer_Vision-green)
![Face Recognition](https://img.shields.io/badge/Face_Recognition-AI-red)
![License](https://img.shields.io/badge/License-MIT-yellow)

An AI-powered Smart Attendance System built using **Flask**, **OpenCV**, and **Face Recognition** that automates attendance management for educational institutions. The system provides secure role-based access for **Admin**, **Faculty**, and **Students**, enabling real-time facial recognition attendance with interactive analytics and AI-generated insights.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/Smart-AI-Attendance-System.git

cd Smart-AI-Attendance-System
```

---

## Create Virtual Environment

### Windows

```bash
python -m venv venv

venv\Scripts\activate
```

### Linux / macOS

```bash
python3 -m venv venv

source venv/bin/activate
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Run the Application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
